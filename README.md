# Convite de casamento — Doce Cabana

Página estática pronta para publicação no GitHub Pages ou Vercel.

## Publicar na Vercel
1. Envie esta pasta para um repositório no GitHub.
2. Na Vercel, clique em **Add New > Project**.
3. Importe o repositório.
4. Framework Preset: **Other**.
5. Clique em **Deploy**.

Não precisa instalar dependências nem executar comandos de build.
